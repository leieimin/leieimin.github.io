# [恋爱四重奏之靠近时的温度](https://leieimin.github.io/)

同居第三个月，他们发现了彼此最奇怪的习惯。

她挤牙膏一定要从最底部开始，规规矩矩向上推；而他总是随手一捏，哪里方便从哪里来。第一次发现时，两人对着洗漱台上的两支牙膏笑了足足五分钟。

共享空间意味着不断发现这些微小的差异，并学习如何让它们共存。衣柜从“你的”和“我的”变成了“我们的”，书架上的书开始混在一起，连冰箱里的食物也分不清最初是谁买回来的。

深夜，她常被他的键盘声吵醒——他总在凌晨灵感爆发。起初她会抱怨，后来干脆买了副耳塞。而他发现她看电视时一定要开着字幕，哪怕完全听得懂，说这样“更有安全感”。

这些细节起初是新奇，后来偶尔令人烦躁，最终却变成了生活背景中不可或缺的纹理。

真正的亲密不是永远和谐，而是在看见对方最真实甚至最不讨喜的一面后，依然选择靠近。

某个加班的雨夜，她发烧了。他提前结束会议回家，推开门看见她蜷缩在沙发上，脸色潮红。测体温、喂药、用湿毛巾敷额头，动作生疏却温柔。后半夜她退烧了，却抓着他的手不肯放。

“我梦到你了，”她声音沙哑，“在梦里你也这么着急。”

他没说话，只是把她的手握得更紧了些。

清晨，她在粥的香气中醒来。厨房里，他正对照手机食谱手忙脚乱。粥有点糊，咸菜切得大小不一，但她吃得一滴不剩。

“好吃吗？”

“难吃死了。”她笑着，眼眶微红，“下次还是我来吧。”

那一刻他们明白，爱不是盛大仪式上的誓言，而是病中一碗煮糊的粥，是深夜保留的一盏灯，是愿意为对方改变早已固化的习惯。
